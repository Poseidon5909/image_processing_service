# image_service

Minimal image processing service layout.

Project structure

```
image_service/
├── app/
│   ├── main.py
│   ├── db.py
│   └── __init__.py
├── .env
├── requirements.txt
└── README.md
```

Quick start

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app (adjust as needed):

```bash
python -m app.main
```
